                        Comparable parliamentary corpus
                                ParlaMint-IT 2.1
                              TEI encoded version
                         
         Citation, documentation, download, and licence available from
                        http://hdl.handle.net/11356/1432

This directory contains the ParlaMint/TEI encoded version of the ParlaMint-IT
2.1 corpus. The root file ParlaMint-IT.xml contains the corpus teiHeader and
XIncludes of the component files, which have the form ParlaMint-IT_<suffix>.xml.
